# stock-price-prediction-sentiment-analysis

##  Project Overview
This project integrates Twitter sentiment analysis with stock market data to improve prediction performance.

##  Technologies Used
- Python
- Apache Spark
- VADER Sentiment Analysis
- SARIMAX
- LSTM

##  Approach
- Data pipeline using Spark
- Sentiment extraction from Twitter data
- Time-series modeling using SARIMAX and LSTM
- Performance comparison using RMSE and R²

##  Results
LSTM demonstrated improved performance over traditional time-series models.

##  Key Skills Demonstrated
Big data processing, sentiment analysis, time-series forecasting, model comparison.
